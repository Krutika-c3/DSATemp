Majority Elements (> N/3 times)

Problem Statement: Given an integer array of size n, find all elements that appear more than floor(n/3) times.

Observation:

  • At most, there can be 2 majority elements.
  • Reasoning: If there were 3 elements, each appearing (n/3 + 1) times, their total occurrences would be 3 * (n/3 + 1) = n + 3, which exceeds the array size n.

Approaches

1. Brute Force Approach

Logic:

  1. Initialize an empty list to store the answers.
  2. Iterate through every element i in the array.
  3. Optimization: If the result list already contains 2 elements, break immediately.
  4. Check if the current element is already in the result list. If so, skip it to avoid duplicates.
  5. Run an inner loop to count the frequency of arr[i] in the entire array.
  6. If the count is greater than n/3, add it to the result list.

Algorithm:

  1. Initialize list = [].
  2. Loop i from 0 to n-1.
  3. Inside, if list is not empty and list[0] == arr[i], continue (duplicate check).
  4. Initialize count = 0.
  5. Loop j from 0 to n-1:
  • If arr[j] == arr[i], count++.
  1. If count > n/3, add arr[i] to list.
  2. If list.size() == 2, break.
  3. Return list.

Complexity:

  • Time Complexity: O(N^2) (Nested loops).
  • Space Complexity: O(1) (The result list stores at most 2 elements).

Java Code:

import java.util.*;

public class Solution {
    public List<Integer> majorityElement(int[] v) {
        int n = v.length;
        List<Integer> list = new ArrayList<>();

        for (int i = 0; i < n; i++) {
            // Optimization: If we already found 2 elements, stop.
            if (list.size() == 0 || list.get(0) != v[i]) {
                int count = 0;
                for (int j = 0; j < n; j++) {
                    if (v[j] == v[i]) {
                        count++;
                    }
                }

                if (count > n / 3) {
                    list.add(v[i]);
                }
            }
            if (list.size() == 2) break;
        }

        return list;
    }
}

2. Better Approach (Using Hashing)

Logic:

  1. Use a HashMap to store the frequency of each element.
  2. Calculate the threshold minRequired = (n / 3) + 1.
  3. Iterate through the array and update the map.
  4. While iterating, if an element's count reaches the threshold (minRequired) exactly, add it to the result list.
  • This ensures we add the element exactly once without needing a second pass over the map or a separate set for uniqueness.
  1. If the list size reaches 2, break immediately.

Algorithm:

  1. Initialize map and list.
  2. Calculate mini = (n / 3) + 1.
  3. Loop through the array:

  • Increment frequency of current element in map.

  • If map.get(element) == mini:

  • Add element to list.

  • If list.size() == 2, break.

  1. Sort the list (if required by output format) and return.

Complexity:

  • Time Complexity: O(N) (Iterating through array).
  • Space Complexity: O(N) (Storing unique elements in the map).

Java Code:

import java.util.*;

public class Solution {
    public List<Integer> majorityElement(int[] v) {
        int n = v.length;
        List<Integer> list = new ArrayList<>();
        Map<Integer, Integer> map = new HashMap<>();

        // Threshold required
        int mini = (int)(n / 3) + 1;

        for (int i = 0; i < n; i++) {
            int value = map.getOrDefault(v[i], 0);
            map.put(v[i], value + 1);

            // Check if it reached the minimum requirement just now
            if (map.get(v[i]) == mini) {
                list.add(v[i]);
            }
            
            // Optimization: Stop if we found 2 numbers
            if (list.size() == 2) break;
        }
        
        Collections.sort(list); // Optional: Sort result
        return list;
    }
}

3. Optimal Approach (Extended Boyer-Moore Voting Algorithm)

Logic:
Since there can be at most 2 majority elements, we maintain two candidates (el1, el2) and two counters (cnt1, cnt2).

  1. Candidate Selection (First Pass):
  • If cnt1 is 0 and the current number is not el2, pick the current number as el1 and set cnt1 = 1.
  • Else if cnt2 is 0 and the current number is not el1, pick the current number as el2 and set cnt2 = 1.
  • Else if current number equals el1, increment cnt1.
  • Else if current number equals el2, increment cnt2.
  • Else (current number matches neither), decrement both cnt1 and cnt2.
  1. Verification (Second Pass):
  • The first pass only gives potential candidates. We must verify if they actually appear more than n/3 times.
  • Reset counters and count actual occurrences of el1 and el2.
  • Add to result if condition is met.

Algorithm:

  1. Initialize cnt1 = 0, cnt2 = 0.
  2. Initialize el1 = Integer.MIN_VALUE, el2 = Integer.MIN_VALUE.
  3. Pass 1: Loop through array nums:
  • If cnt1 == 0 AND nums[i] != el2: cnt1 = 1, el1 = nums[i].
  • Else If cnt2 == 0 AND nums[i] != el1: cnt2 = 1, el2 = nums[i].
  • Else If nums[i] == el1: cnt1++.
  • Else If nums[i] == el2: cnt2++.
  • Else: cnt1--, cnt2--.
  1. Pass 2: Reset cnt1 = 0, cnt2 = 0. Loop through array nums:
  • If nums[i] == el1: cnt1++.
  • If nums[i] == el2: cnt2++.
  1. Check threshold mini = (n / 3) + 1.
  2. If cnt1 >= mini, add el1 to list.
  3. If cnt2 >= mini, add el2 to list.
  4. Sort and return list.

Complexity:

  • Time Complexity: O(N) + O(N) = O(2N) which simplifies to O(N) (Two linear passes).
  • Space Complexity: O(1) (Only 4 fixed variables used).

Java Code:

import java.util.*;

public class Solution {
    public List<Integer> majorityElement(int[] v) {
        int n = v.length;
        int cnt1 = 0, cnt2 = 0;
        int el1 = Integer.MIN_VALUE;
        int el2 = Integer.MIN_VALUE;

        // Pass 1: Find potential candidates
        for (int i = 0; i < n; i++) {
            if (cnt1 == 0 && v[i] != el2) {
                cnt1 = 1;
                el1 = v[i];
            } else if (cnt2 == 0 && v[i] != el1) {
                cnt2 = 1;
                el2 = v[i];
            } else if (v[i] == el1) {
                cnt1++;
            } else if (v[i] == el2) {
                cnt2++;
            } else {
                cnt1--;
                cnt2--;
            }
        }

        // Pass 2: Verify candidates
        List<Integer> ls = new ArrayList<>();
        cnt1 = 0;
        cnt2 = 0;
        
        for (int i = 0; i < n; i++) {
            if (v[i] == el1) cnt1++;
            if (v[i] == el2) cnt2++;
        }

        int mini = (int)(n / 3) + 1;
        if (cnt1 >= mini) ls.add(el1);
        if (cnt2 >= mini) ls.add(el2);

        Collections.sort(ls);
        return ls;
    }
}
